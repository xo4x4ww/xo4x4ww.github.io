# Pixel Portfolio

Персональное портфолио в пиксельном стиле.

## Настройка

1. Откройте `config.js`
2. Измените все настройки под себя
3. Добавьте свои навыки в `SKILLS.items`
4. Добавьте свои ачивки в `ACHIEVEMENTS.items`
5. Обновите соц-ссылки в `SOCIAL`

## Структура

- `index.html` - основной файл
- `config.js` - ВСЕ настройки сайта
- `style.css` - стили с пиксельным дизайном
- `script.js` - логика и функциональность
- `assets/images/` - ваши изображения

## Запуск

Просто откройте `index.html` в браузере!
